• 3 - 1. 畫面介紹

    • 錢包：

      實際擁有及儲値點數顯示處。

    • 大賞燈：

      大獎(BB)及小獎(RB)開獎紀錄、運轉次數。

    • 遊戲主錢包：

      總分板。

    • 機台錢包：

      機台本身點數也是主要遊戲扣點處; 最多 50枚。

    • 得獎資訊：

      開獎顯示處。

    • 遊戲得分：

      遊戲得分分數顯示。

  • 3 - 2. 按鈕介紹

    模式：自動 / 手動。

    自動：機台自動RUN遊戲；如需停止，再點擊一下便可停止。

    手動：手動點擊遊戲開始及暫停。

    • 保留：

      遊戲機台做保留，其他人無法點選。

    • 離開：

      結束遊戲。

    • 重整：

      更新遊戲版面。

    • 看表：

      機台歷史紀錄。

    • 履歷：

      當局遊戲紀錄(保留機台會延續累計；離開機台便重新統計)。

    • 開分：

      點數轉入進遊戲機台中，最低限制 100。

    • 洗分：

      主錢包點數轉入錢包中，最低限制 100。

    • 投幣：

      點選投幣按鈕便會從主錢包將點數轉入機台錢包。

    • 退幣：

      機台錢包點數轉到主錢包上。

  • 3 - 3. 執行開分

    Step.1 進入遊戲後先執行開分，轉入點數送出後才可進行遊戲

    • 遊戲主錢包：

      遊戲總分板。

    • 機台錢包：

      機台本身點數也是主要遊戲扣點處; 最多 50枚開分 100主錢包及機台錢包分別會値入 50 。

    每當進行遊戲時便會使用 3 枚代幣。當出現無滿足開分點數時，為遊戲機台本身已有存留代幣所導致。(以圖中情況來說，由於機台本身已留有1 枚代幣，故主錢包便會扣 2 枚，而顯示 48)。

  • 3 - 4. 選擇模式

    Step.2 進入遊戲後選擇模式

    執行遊戲可自動或手動方式執行

    • 手動模式
    • 自動模式

  • 3 - 5. 機台錢包

    Step.3 投入金額換算遊戲錢包進行遊戲
    當機台錢包為 0 時，點選投幣按鈕便會從主錢包將點數轉入機台錢包。

  • 3 - 6. 洗分換算

    Step.4 點選洗分換算金額轉入自身錢包
    再點選洗分按鈕，將主錢包點數轉至自己錢包中。

  • 3 - 7. 保留遊戲

    Step.5 需保留遊戲可點選左上角中的保留按鈕

    目的： 讓其他人無法點選。

  • 3 - 8. 結束遊戲

    Step.6 若要結束遊戲，只需點選左上角中的離開按鈕即可

    如沒有選擇離開，系統將自動保留。
